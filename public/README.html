EpointGPT 公益注册站 - 浏览器插件版

🌟 背景说明：随着 OpenAI 风控策略的不断加强，对注册 IP 和域名邮箱的要求不断变高。以往的协议注册方式基本全面失效（除非氪金）。当前免费且广为人知能批量注册的邮箱，几乎只有 163、2925、duck 等少数几家。建议抓住最后的红利期“白嫖”。当然也有可能明天就g了。
🛑 免责声明：本项目旨在建立新点公益站仅供内部使用，共同维护共同使用，用多久算多久，不提供任何稳定运行的承诺。
⚠️ 重要提示： 当前 Web 端的自动注册功能暂不可用（生态没有插件好bug搞不动了），请统一使用本浏览器插件（Extension）模式进行注册。

📥 安装与获取

请下载：epoint-gpt-autoreg-extension.zip
下载并解压后，打开浏览器的扩展程序管理页面，开启“开发者模式”后，点击加载已解压的扩展程序，并选中刚才解压出来的目录。

推荐浏览器：插件只在 Chrome 环境下进行过测试，建议使用 Google Chrome 浏览器。

注：如果你使用 Chrome 办公，建议在 无痕模式中 (Incognito) 开启并运行该插件（扩展管理里勾选“在无痕模式下启用”），否则可能影响你的正常办公。

📖 使用说明

1. 邮箱提供商

  1. 邮箱选项：默认选择 2925 即可（只测过这个）。
  2. 账号密码：主邮箱地址和密码，就是你登录 2925 平台的账号密码。

2. CPA配置

  1. CPA地址和密钥：该配置默认我的中转站地址，初衷是一起管理一起用。若你自己部署了后端的平台，也可以配置为你自己的。

❓ 可能遇到的问题 (FAQ)

  • 1. 2925 收不到验证邮件？
    大概率是梯子节点或网络连通问题导致，可以尝试切换不同地区的干净的节点。

  • 2. 2925 邮箱报错子邮箱达到上限？
    2925 平台对短时间创建的子邮箱数量有限制，遇到该问题后只需要等待大概半小时左右即可恢复。
